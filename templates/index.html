Help & User Guide

Getting Started

Watch Media Server is a Netflix-style media streaming application that organizes and streams your personal media collection.

Media Organization

Organize your media files in the following folder structure:

  • Movies/ - Place all your movie files here
  • TV Shows/ - Place all your TV show files here
  • Kids/ - Place kids-friendly content here
  • Music Videos/ - Place music videos here

Playing Media

  1. Click on any media thumbnail to open the player
  2. Use the video controls to play, pause, seek, and adjust volume
  3. Click the audio-only button for audio-only playback
  4. Click the X button or press Escape to close the player

Supported Formats

MP4, MKV, AVI, MOV, WMV, FLV, WebM

About Watch Media Server

Project Overview

Watch Media Server is a modern, Netflix-style media streaming application designed to organize and stream your personal media collection.

Key Features

  • Netflix-style user interface
  • Automatic media library scanning
  • Smart categorization (Movies, TV Shows, Kids, Music Videos)
  • Real-time streaming with multiple format support
  • Audio-only playback option
  • Responsive design for all devices
  • Dark/Light theme support

Version Information

Version: 1.3.0
Build Date: 2025-09-14
Technology: Python Flask, HTML5, CSS3, JavaScript

Authorship

Developed with AI assistance (Claude Sonnet 4) for full-stack development, architecture design, and implementation.

About Watch Media Server

Project Overview

Watch Media Server is a modern, Netflix-style media streaming application designed to organize and stream your personal media collection. Built with Python Flask and featuring a responsive web interface, it provides an intuitive way to manage and enjoy your movies, TV shows, and other media content.

Key Features

  • Netflix-style user interface
  • Automatic media library scanning
  • Smart categorization (Movies, TV Shows, Kids, Music Videos)
  • Real-time streaming with multiple format support
  • Responsive design for all devices
  • Dark/Light theme support
  • RESTful API for integration
  • Docker containerization

Technical Details

Technology Stack

  • Backend: Python 3.13, Flask, SQLite
  • Frontend: HTML5, CSS3, JavaScript (ES6+)
  • Media Processing: FFmpeg, MediaInfo
  • Containerization: Docker
  • Database: SQLite with automatic migrations
  • Styling: Custom CSS with Font Awesome icons

Architecture

The application follows a modular architecture with separate services for media management, streaming, and user interface. The backend provides a RESTful API while the frontend delivers a modern, responsive user experience.
